怎么发链接给朋友测

这是纯静态网页，不需要后端。把这个 HTML 文件部署到静态托管平台后，别人点链接就能直接打开测试。最省事的是 Netlify 或 Vercel。

GitHub Pages

把文件命名为 index.html，放进 GitHub 仓库根目录，开启 Pages，就能得到公开链接。适合你后面继续迭代版本。

仓库里至少保留： index.html GitHub -> Settings -> Pages -> Deploy from a branch -> main / root

Vercel

最适合你这种单页网页。导入 GitHub 仓库或者直接上传目录，Vercel 会自动识别成静态站点并上线。

本地文件夹结构： my-sbti-site/ index.html 上传到 GitHub 后，Vercel 里点 New Project 导入即可。

Netlify

最省事。你甚至可以把只有一个 index.html 的文件夹直接拖进 Netlify 部署页面，它会立即给你一个链接。

部署最小结构： my-sbti-site/ index.html 拖拽整个文件夹到 Netlify Drop 页面即可。
